# Calculator App

This project is a simple calculator app built with HTML, CSS, and JavaScript. It provides basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Basic calculator functionality
- Responsive design for mobile and desktop
- Clean user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

To view the project locally:

1. Clone the repository:

    ```bash
    git clone https://github.com/AdityaBN271/skills-introduction-to-github/new/Rd_infro_technology/calculator
    ```

2. Navigate to the project folder:

    ```bash
    cd calculator
    ```

3. Open the `index.html` file in your browser.
